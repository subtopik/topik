# Getting Started with Topik

Topik compiles your markdown content into structured, validated resources.

## Installation

Add `@topik/astro` to your Astro project and point it at your content directories.

## Writing Guides

Create a `collection.yaml` to define your guide collection, then add markdown files alongside it.

Each markdown file becomes a guide with its frontmatter mapped to structured metadata.
