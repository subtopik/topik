# Writing Markdown for Topik

Topik supports standard markdown with YAML frontmatter.

## Frontmatter

Use YAML frontmatter to define metadata like title, description, authors, and tags.

## Content

Write your content using standard markdown syntax. Topik preserves your content as-is and passes it through to your rendering pipeline.
